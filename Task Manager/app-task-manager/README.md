## Description
A task manager frontend built for personal dev-lab learning. This project is designed for build UI for Nest-Task-manager APIs.

## Technologies
React and TypeScript

## Project setup

```bash
$ npm install
```

# Compile and run the project

```bash
# development
$ npm run start

# build
$ npm run build

# Eject
$ npm run eject
```

# Run tests

```bash
# tests
$ npm run test
```
